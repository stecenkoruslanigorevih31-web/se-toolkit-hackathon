Context

👤 End-User

Students and professionals who need daily motivation and inspiration tailored to their current situation.

❌ Problem

Generic motivational quotes don't match the user's specific field (sport, study, career) or current mood.

💡 Product Idea

AI-powered web app where users select a field and mood to receive personalized motivational sentences, save favorites, and build an inspiration library.

Implementation

  • Stack: Node.js + Express.js backend, SQLite database, vanilla JS frontend
  • AI: OpenRouter free API (Llama/Gemma models)
  • UI: Responsive three-panel layout (History, Controls, Favorites)

Version 1

Core generation feature with field/mood selection, history, favorites, and persistent SQLite storage.

Version 2

Docker Compose deployment, Nginx reverse proxy, Jest test suite (9 tests), comprehensive README, MIT license.
