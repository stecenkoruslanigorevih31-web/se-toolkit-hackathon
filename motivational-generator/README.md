# Мотивационный Генератор 🚀

Веб-приложение для генерации мотивационных текстов с помощью AI Qwen.

## 🎯 Функциональность

- 🤖 Генерация уникальных мотивационных текстов через Qwen API
- 🎨 Красивый и отзывчивый интерфейс
- ⚡ Быстрая генерация текстов
- 🔒 Безопасное хранение API ключа на сервере

## 📋 Требования

- Node.js (версия 14+)
- npm
- Qwen API ключ

## 🚀 Запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Конфигурация
Создайте файл `.env` в корне проекта:
```
QWEN_API_KEY=ваш_api_ключ_здесь
PORT=3000
```

### 3. Запуск сервера
```bash
npm start
```

Откройте браузер и перейдите на `http://localhost:3000`

## 🔧 Технический стек

- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: Qwen API (Alibaba)

## 📝 Использование

1. Откройте веб-приложение в браузере
2. Нажмите кнопку "Получить Мотивацию"
3. Дождитесь генерации текста
4. Вдохновляйтесь! ✨

## 🐛 Решение проблем

- **Ошибка API**: Проверьте корректность API ключа в файле `.env`
- **Порт занят**: Измените PORT в `.env` на другой порт
- **Модули не установлены**: Запустите `npm install`

## 📦 Развертывание на VM

Для развертывания на виртуальной машине:

1. Скопируйте все файлы проекта на VM
2. Установите Node.js на VM
3. Выполните `npm install`
4. Создайте `.env` файл с API ключом
5. Запустите `npm start`
6. Настройте nginx/apache как reverse proxy (опционально)
