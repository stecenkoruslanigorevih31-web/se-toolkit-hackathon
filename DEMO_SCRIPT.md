# 🎬 Demo Video Script (max 2 minutes)

## Preparation
1. Start the server: `npm start`
2. Open http://localhost:3000 in your browser
3. Start screen recording with voice (OBS, Loom, or built-in Windows recording)

---

## Recording Script (read aloud)

### [0:00-0:15] Introduction
> "This is Motivational Generator — an AI-powered web app that generates personalized motivational quotes.
> The user selects a field and a mood, and the AI creates a unique motivational sentence tailored to their choice."

### [0:15-0:45] Generation
> "On the left — the history of all generated sentences. In the center — the control panel. On the right — saved favorites.
> I select the field: Career & Success. I select the mood: Unstoppable & Driven.
> I click Generate Motivation...
> The AI generated a motivational sentence that matches my selected field and mood."

### [0:45-1:15] Favorites and History
> "I like this quote — I click Save to Favorites.
> It now appears in the Favorites panel on the right.
> The History panel on the left shows all generated sentences with timestamps and field/mood tags.
> I can remove a quote from favorites if I no longer need it."

### [1:15-1:45] Technology Stack
> "The backend is built with Node.js and Express. The SQLite database persists data between restarts.
> The AI runs through the free OpenRouter API using Llama and Gemma models.
> The application is fully containerized with Docker Compose and an Nginx reverse proxy."

### [1:45-2:00] Conclusion
> "The code is on GitHub, the app is deployed on a virtual machine and accessible via a public link.
> Thank you for watching."

## Recording Tools

### Free options:
1. **OBS Studio** — full-featured, free, records screen + voice
   - Download: https://obsproject.com
   - Settings → Output → Recording format: MP4

2. **Loom** — browser-based, automatically generates a shareable link
   - https://www.loom.com
   - Free up to 5 minutes

3. **Windows built-in recording** — Win + G → Record
   - Already installed on Windows

### How to save:
- Record in MP4 format
- Upload to Google Drive / YouTube (unlisted) / Loom
- Paste the link into the presentation
